NC News API

A RESTful API for accessing NC News articles, comments, topics and users.

Endpoints

  • GET /api/topics
  • GET /api/users
  • GET /api/articles
  • GET /api/articles/:article_id
  • GET /api/articles/:article_id/comments
  • POST /api/articles/:article_id/comments
  • PATCH /api/articles/:article_id
  • DELETE /api/comments/:comment_id

GET /api/topics

GET /api/topics

GET /api/articles

Responds with a list of all available articles.
Takes optional queries.

Queries

  • sort_by (optional): column to sort by
  • order (optional): asc or desc
  • topic (optional): filter articles by topic

Example Request

GET /api/articles?sort_by=votes&order=desc&topic=coding

Example Result

            {
                "articles": [
                    {
                        "article_id": 1,
                        "title": "Running a Node App",
                        "topic": "coding",
                        "author": "jessjelly",
                        "votes": 0,
                        "comment_count": 8,
                        "created_at": "2020-11-07T00:00:00.000Z"
                    }
                ]
            }

POST /api/articles/:article_id/comments

Add a comment to an article on the given article_id.
The article_id must be an number, or the request will return an error.
The body of the post must be in JSON format and contain an author and a body key.
The author value must be an author that already exists in the database, and must be a string.

Example Request

POST /api/articles/2/comments
            body = {
                "author": "grumpy19",
                "body": "This is a test comment"
            }

Example Result

            {
                "comment": {
                    "comment_id": 304,
                    "article_id": 4,
                    "body": "This is a test comment",
                    "votes": 0,
                    "author": "grumpy19",
                    "created_at": "2026-02-09T22:16:46.229Z"
                }
            }

PATCH /api/articles/:article_id

Updates an articles vote count on a given article based on article_id
The article_id must be an number, or the request will return an error.
The body of the post must be in JSON format and contain a key of inc_votes.
The inc_votes value must be an integer.

Example Request

PATCH /api/articles/2
            body = {
                "inc_votes":4
            }

Example Result

            {
                "article": [
                    {
                        "article_id": 2,
                        "title": "The Rise Of Thinking Machines: How IBM's Watson Takes On The World",
                        "topic": "coding",
                        "author": "jessjelly",

Most recently, IBM has announced several new partnerships which aim to take things even further, and put its cognitive capabilities to use solving a whole new range of problems around the world.",

DELETE /api/comments/:comment_id

Deletes a comment by given comment_id.

Example Request

DELETE /api/comments/2

Will return a 204 with no body in the response.

NC News API - built with Node.js, Express, and PostgreSQL
